# v1 开发计划（Phase 划分）

> 依据：`技术栈选型.md`（栈已定 C#/WPF）、`功能需求.md`（FR 编号）、`项目规格.md`。
> 约束：单 exe（双模式守护）、仅 Windows、内存目标 <80MB（自包含发布可能放宽，见 `技术栈选型.md` §4）、零 WebView2、防误杀。
> 各功能模块技术选型见 `技术栈选型.md` §6。

## 阶段总览（依赖链）

```
P0 脚手架+守护 ─► P1 渲染基座+多层+画布+单原子 ─► P2 原子全集+属性三元组+公式引擎
                                                   │
                                                   ▼
P3 网格+小组件+页面 ─► P4 数据接入(si/mi/mu/ai/an) ─► P5 行为系统 ─► P6 背景+设置+持久化闭环+打磨
```

- 每阶段结束产出**可运行 exe**（P2 中期起即带可见功能）。
- **MVP 切点 = P0 + P1 + P2**：一个带实时公式时钟的覆盖层，证明全链路（窗口→画布→原子→公式→持久化→守护）跑通。

---

## P0 · 脚手架 + 守护进程
**目标**：一个能双击运行的 exe + 守护 + 覆盖窗口空壳。
**交付**
- .NET 8 项目（`net8.0-windows`），单 exe `overlay.exe`（win-x64 发布）。
- WPF 原生窗口 + Win32 互操作脚手架（参考已验证的 Win32 定位思路，无 WebView2）。
- 双模式入口：`Main` 解析 `--guardian`；UI 启动即 `Process.Start` 拉起看门狗兄弟进程。
- 互相看门：命名管道 `\\.\pipe\overlay-guardian` 心跳 + 命名事件 `Global\overlay-clean-shutdown`（主动退出不重启）+ 互持 `Process` 句柄 `WaitForExit` 重拉。
- 注册表 `HKCU\...\Run` 自启（带开关，可取消）。
- 覆盖窗口：P/Invoke 设全屏（**非全局置顶**，盖桌面图标、居普通应用之下）+ **任务栏系统自动隐藏（Auto-hide）铺 `SystemParameters.WorkArea` 不预留**（弹出浮于其上）+ **输入拦截**（空白吸收、交互元素 `IsHitTestVisible=True`；方案 B `WS_EX_TRANSPARENT`+命中测试备选，见 `渲染架构设计.md` §3.1 / `技术栈选型.md` §6）。
- `RegisterApplicationRestart` 崩溃兜底。
**验收**：双击 → 全屏覆盖层出现，原生桌面图标/壁纸隐藏且不可交互；任务栏可见可点；杀 UI 进程 → 看门狗秒级拉起；自启生效；崩溃后系统自重启。
**技术**：WPF + Win32 P/Invoke、RegisterApplicationRestart、Process.Start 守护拉起。
**关联**：覆盖核心（项目规格 §1/§4.2）、守护（技术栈选型 §3）、任务栏 Auto-hide 不预留（项目规格 §3）。

## P1 · 渲染基座 + 多层 + Px 画布 + 单原子
**目标**：渲染管线与坐标系跑通，多层模型，Px 画布可放一个 Text 原子（静态值）。
**交付**
- WPF `Window`(XAML) + 多层模型（Layer：`Panel.ZIndex` / `Opacity` / 9 项可配骨架，v1 先落 ZIndex + Opacity + Enabled）。
- 三套坐标系：Screen / Canvas(px) / Grid(cell)（功能需求 §3.0）。
- `Atom` 抽象类 + `ATOM_REGISTRY` 运行时注册；渲染分发。
- Text 原子（静态字符串）+ Px 画布拖拽定位 + 标尺/吸附参考线。
- 临时 JSON 存一个原子的 x/y/w/h。
**验收**：画布上放一个 Text，可拖动、px 定位、刷新后保留。
**技术**：WPF Canvas 绝对坐标、C# Atom 抽象类、System.Text.Json。
**关联 FR**：FR-CANVAS-01~05、FR-ATOM-01~06/18、FR-LAYER-*。

---

## P2 · 原子全集 + 属性三元组 + 公式引擎
**目标**：六类原子齐 + 属性三元组（静态/gv/公式）+ 公式引擎 19 函数 + gv + 持久化。
**交付**
- 六类原子：Text(含 CurvedText) / Shape(渐变填充) / Icon / Image / Progress / Container(Overlap/Stack/Series)。
- 属性三元组：`Static` / `Global(gv)` / `Formula($...$)`，由 toggle 控制。
- 公式引擎：Lexer→Parser→AST→FunctionRegistry→DataProvider；v1 必做 19 函数（df tf si bi gv if tc mi mu + ts tu tz uc re fl dp rng）。
- 运算符 `= > >= < <= != ~=`、`+` 拼接、`~=` 正则。
- 全局变量 gv：类型化（NUMBER/TEXT/COLOR/FONT/...），跨层共享，改值即增量重算。
- 脏标记增量重算；求值错误回退静态值不崩。
- JSON 持久化：atom（x/y/w/h/z + 属性三元组）+ layer 配置。
**验收**：Text 上 `$df(hh:mm:ss)$` 实时跑；`$if(bi(level)=100,"满",bi(level))$` 正确；gv 改值全层刷新；配置存盘重开一致。
**技术**：手写 Lexer/Parser、DateTime、System.Text.RegularExpressions、Random、System.Text.Json。
**关联 FR**：FR-ATOM-* 全、公式引擎（见 `公式引擎设计.md`/`公式函数参考.md`）、v1 必做 19 函数（公式函数参考 §4）。

## P3 · 网格 + 小组件 + 页面
**目标**：网格（gcd 四档）+ 小组件（原子组合入库）+ 多页面切换。
**交付**
- 网格：gcd 间距推导，四档 {20,40,60,120}（1920×1080），FR-GRID-08~12；cell→px。
- 小组件：`WIDGET_REGISTRY`，原子树打包 → 小组件，拖入 cell，跨行列 resize，入库复用。
- 页面：`ObservableCollection<Page>`，每页独立网格+背景，切换（指示/滚轮/快捷键）+ 淡入淡出（Storyboard），上限默认 9。
**验收**：把一组原子打包成「时钟小组件」拖入网格 cell；多页切换无闪烁。
**技术**：C# gcd 网格数学、WPF UniformGrid 单元布局、页面模型（ObservableCollection<Page>）。
**关联 FR**：FR-GRID-*、FR-WIDGET-01~07、页面（§6）。

---

## P4 · 数据接入（si / mi / mu / ai / an）
**目标**：系统指标/媒体/启动坞数据源接通，公式出真实数据。
**交付**
- `System.Diagnostics.PerformanceCounter`(PDH) → `si()`（CPU/内存/网速/磁盘/进程）。
- `Windows.Media.Control`(SMTC, WinRT) → `mi()`/`mu()`（播放状态/曲名/控制）。
- 启动坞：开始菜单 `.lnk` 扫描（`IShellLink`）+ UWP `PackageManager` → `ai()`/`an()` 枚举已装程序。
- 数据接入公式 Provider（闭环 P2 引擎）。
**验收**：`$si(cpu)$` / `$mu(title)$` 实时；启动坞列出已装程序并可启动。
**技术**：System.Diagnostics.PerformanceCounter(PDH)、Windows.Media.Control(SMTC)、IShellLink + PackageManager（启动坞）。
**关联 FR**：具体部件（§6 系统监控/媒体/启动坞）、si/mi/mu/ai/an 函数。

---

## P5 · 行为系统（动画 / 触发器 / 按钮）
**目标**：动画、触发器、按钮行为完整。
**交付**
- 动画：WPF Storyboard / 属性动画（进入/退出/循环：淡入·滑动·缩放·旋转，缓动/延迟）。
- 触发器：显示层/点击/悬停/空闲(`GetLastInputInfo`)/定时/系统事件(锁屏·电量低·网络)。
- 按钮行为：切层显隐·切页·运行程序(`Process.Start`)·发快捷键(`SendKeys`)·触发动画·开 URL(`Process.Start`)·执行脚本。
**验收**：点启动坞图标启动程序；空闲 N 秒触发显隐；进入动画淡入。
**技术**：WPF Storyboard、Win32 事件(GetLastInputInfo)、SendKeys、Process.Start。
**关联 FR**：FR-BEH-01~10、行为系统（§1.7）。

---

## P6 · 背景 + 设置面板 + 持久化闭环 + 打磨
**目标**：背景渲染、设置面板、完整持久化、体积/性能达标。
**交付**
- 背景：图片(填充/适应/平铺/居中/拉伸)/纯色/线性·径向渐变/`dwmapi`(`DwmEnableBlurBehindWindow`) 亚克力毛玻璃；每页独立背景 + 切换过渡。
- 设置面板：`Hardcodet.NotifyIcon.Wpf` 托盘唤起 WPF 设置窗（快捷键/网格间距四档/背景/自启/多屏）；配置导入导出。
- 持久化闭环：layers + behavior + widgets + atoms + pages 全量 JSON。
- 打磨：dotnet publish 体积核对（自包含裁剪 vs 框架依赖，见 `技术栈选型.md` §4）；空闲 CPU ~0；帧 <16ms；崩溃/被杀自动恢复；单屏→单实例验证。
**验收**：设置改网格间距即时生效；配置导出导入无丢失；Release exe 体积达标（<80MB 或按 §4 取舍放宽）；常驻内存达标；杀进程/崩溃均自恢复。
**技术**：ImageBrush/BitmapImage、dwmapi(DwmEnableBlurBehindWindow)、Hardcodet.NotifyIcon.Wpf、System.Text.Json、dotnet publish 体积核对。
**关联 FR**：背景（§4）、设置/控制层（§7）、非功能（§8，体积/性能）。
