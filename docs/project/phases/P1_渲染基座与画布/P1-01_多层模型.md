# P1-01 多层模型

> Phase：P1 · 渲染基座 + 多层 + Px 画布 + 单原子
> 上游：`v1开发计划.md` P1、`技术栈选型.md` §6（多层系统）、`功能需求.md` FR-LAYER-01~03
> 关联：`P1-02_坐标系.md`、`P1-03_Atom抽象与注册.md`、`P0-01_覆盖窗口.md`（宿主）

## 目标
建立 `Layer` 抽象与 `LayerStack` 合成机制，验证多层按 z 序叠加；提供三类预设层骨架（壁纸层 / 网格化层 / 原子化 Px 画布层）。v1 先落 **ZIndex + Opacity + Enabled** 三项可配，其余属性骨架预留。

## 范围
**包含**
- `Layer` 类：`ZIndex` / `Opacity` / `Enabled` + `Root`(UIElement 容器)。
- `LayerStack`：持有 `ObservableCollection<Layer>`，按 `ZIndex` 重排 `Panel.ZIndex`，`Opacity` 绑定，`Enabled=False` 折叠层。
- 三类预设层工厂：`WallpaperLayer` / `GridLayer` / `CanvasLayer`（仅结构，内容在后续 Phase）。

**不含**：混合模式（multiply/screen，留 v1.x，`技术栈选型.md` §6.1）；层内具体内容（P2/P3/P6）。

## 关键设计
### 合成
- 宿主 `Grid`(或 `Canvas`) 为层容器；每层一个 `Panel`，`Panel.ZIndex` 绑定 `Layer.ZIndex`。
- `Opacity` 直接绑 `Panel.Opacity`；`Enabled` 控制 `Panel.Visibility`。

### 预设层
```csharp
abstract class Layer {
    public int ZIndex { get; set; }
    public double Opacity { get; set; } = 1.0;
    public bool Enabled { get; set; } = true;
    public Panel Root { get; } = new Grid();
}
class WallpaperLayer : Layer { }   // 仅背景，最底
class GridLayer     : Layer { }   // 网格格点盛放网格部件
class CanvasLayer   : Layer { }   // Px 画布，自由放原子
```

## 接口草图
```csharp
class LayerStack {
    ObservableCollection<Layer> Layers = new();
    void Recompose();              // ZIndex 重排 + 挂载 Root 到宿主
    Layer AddPreset(LayerKind kind);
}
enum LayerKind { Wallpaper, Grid, Canvas }
```

## 技术选型
WPF `UIElement` 树（`Panel.ZIndex` + `Opacity`），`ObservableCollection` 驱动 MVVM 式重排（`技术栈选型.md` §6 多层系统行）。无第三方依赖。

## FR 映射
FR-LAYER-01（多层概念）、FR-LAYER-02（z / 不透明度 / 启用）、FR-LAYER-03（三类预设定义）。

## 验收
三层按 ZIndex 合成、Opacity 生效、Enabled 切换显隐；新增预设层即插入对应 z 位置。

## 依赖与顺序
- 依赖：`P0-01_覆盖窗口.md`（宿主窗口）。
- 被依赖：P1-03 原子渲染挂到 `CanvasLayer.Root`；P2 持久化层配置；P3 网格挂 `GridLayer`。

## 风险 / 开放
- 混合模式暂未实现，层间仅普通 alpha 叠加；如需正片叠底需 `VisualBrush`/自定义 `Brush`，留 v1.x。
- 层数量上限与性能：v1 层数有限（个位数），暂不优化大层树。
