# P2-01 六类原子

> Phase：P2 · 原子全集 + 属性三元组 + 公式引擎
> 上游：`v1开发计划.md` P2、`功能需求.md` FR-ATOM-(全)（六类：Text/Shape/Icon/Image/Progress/Container）、`技术栈选型.md` §6（原子框架）
> 关联：`P1-03_Atom抽象与注册.md`、`P2-02_属性三元组.md`、`P2-03_公式引擎.md`、`P3-02_小组件.md`

## 目标
实现全部 **六类原子**，每类均能经 `Render()` 产出 WPF `UIElement`，并通过 `P1-03` 注册表分发。每类属性支持 `P2-02` 三元组（静态 / gv / 公式），文本类支持公式 `$...$` 内联。

## 范围
**六类**
1. **Text**（含 CurvedText 弧形文字，用 `Glyphs` + `PathGeometry`）：文本内容 / 字体 / 颜色 / 对齐 / 旋转。
2. **Shape**（渐变填充）：矩形 / 椭圆 / 线 / 圆角矩形，填充支持纯色 / 线性·径向渐变。
3. **Icon**：图标字体字形（`Segoe MDL2` / `FontAwesome`）+ 尺寸 / 颜色。
4. **Image**：位图（`BitmapImage` / `ImageBrush`），填充 / 适应 / 平铺 / 居中 / 拉伸（背景模式复用）。
5. **Progress**：进度条 / 圆环，绑定数值（0–100 或 0–1）。
6. **Container**（布局容器）：`Overlap`(重叠) / `Stack`(堆叠) / `Series`(序列)，承载子原子树。

**不含**：数据接入（si/mi/mu/ai/an，P4）；行为动画（P5）；背景专属毛玻璃（P6，但 Image 填充模式可复用）。

## 关键设计
### 类结构
```csharp
class TextAtom   : Atom { /* Text/Family/Size/Color/Rotation; Curved via PathGeometry */ }
class ShapeAtom  : Atom { /* Kind; Fill(纯色/渐变) */ }
class IconAtom   : Atom { /* Glyph(FontAwesome/MDL2 code); Size; Color */ }
class ImageAtom  : Atom { /* Source; Stretch(Fill/Uniform/Title/Center) */ }
class ProgressAtom: Atom { /* Value(三元组); Orientation; Kind(Bar/Ring) */ }
class ContainerAtom: Atom { /* Layout(Overlap/Stack/Series); Children:List<Atom> */ }
```
- 每个原子 `Render()` 返回对应 `UIElement`；`ContainerAtom.Render()` 递归渲染 `Children` 并按布局排列。
- 文本内容（Text/Icon 字形 / Progress 数值 / Shape 标签）支持三元组：`"$df(hh:mm:ss)$"` 解析见 `P2-03`。

### 图标字体
- `Segoe MDL2 Assets`（系统内置）+ `FontAwesome`（NuGet `FontAwesome.Sharp` 或自嵌字体），`技术栈选型.md` §5 白名单。
- 字形以 unicode / 私用区码点表示，`TextBlock` + 指定 `FontFamily` 渲染。

### 弧形文字（CurvedText）
- 用 `PathGeometry` 定义弧线路径，`Glyphs` 沿路径排版（`技术栈选型.md` §5 弧形文字项）。v1 先支持简单弧，复杂路径留 v1.x。

## 技术选型
WPF `TextBlock` / `Path` / `Geometry` / `Image` / `ProgressBar` / `Panel`；图标字体 NuGet（`技术栈选型.md` §6 原子框架 + §5 图标字体）。图表类（如有）自绘，不引 LiveCharts。

## FR 映射
FR-ATOM-01~11（六类 + 注册 + 布局容器 + 弧形文字 FR-ATOM (CurvedText)）；进度 / 图标具体 FR 在 `功能需求.md` 原子章。

## 验收
六类原子均可渲染并挂画布；Text 显示 `$df(hh:mm:ss)$` 实时（接 P2-03）；Progress 绑定 `$bi(level)$` 变动；Container 三布局正确排列子原子。

## 依赖与顺序
- 依赖：`P1-03`(抽象/注册)、`P2-02`(三元组)、`P2-03`(公式)、`P2-04`(gv)。
- 被依赖：P3-02 小组件（原子树打包）、P6 背景（Image 模式复用）。

## 风险 / 开放
- CurvedText 排版精度（字间距沿弧）实现成本较高，v1 可仅支持半圆 / 整圆简化。
- 图标字体版权 / 嵌入体积需守 NFR-01，优先系统 `Segoe MDL2` 免嵌。
