# P2-02 属性三元组

> Phase：P2 · 原子全集 + 属性三元组 + 公式引擎
> 上游：`v1开发计划.md` P2、`功能需求.md` 属性三元组（静态 / gv / 公式）、`技术栈选型.md` §6（原子属性）
> 关联：`P2-01_六类原子.md`、`P2-03_公式引擎.md`、`P2-04_全局变量gv.md`、`P2-05_增量重算与容错.md`

## 目标
每个原子属性支持三种来源，由 **toggle** 控制：
- **Static**：固定字面量。
- **Global(gv)**：引用全局变量（跨层共享）。
- **Formula($...$)**：内联公式，经 `P2-03` 求值。

统一解析为运行期值，支持文本内联（`"$df()$ 文本"`）。

## 范围
**包含**
- `PropertyValue` 三态模型 + 解析器（字面量 / `gv:` 引用 / `$...$` 表达式）。
- 文本内联解析：扫描 `$\$...\$` 片段替换为求值结果。
- 类型约束：属性期望类型（文本 / 数字 / 颜色 / 字体）与三元组输出类型相容校验（宽松 + 回退）。

**不含**：公式引擎实现本身（P2-03）；gv 存储（P2-04）；增量重算调度（P2-05）。

## 关键设计
### 三元组模型
```csharp
enum PropMode { Static, Global, Formula }
abstract class PropertyValue {
    public PropMode Mode { get; }
    public abstract object Resolve(EvalContext ctx);   // ctx 含 gv + 公式引擎
}
class StaticValue  : PropertyValue { public object Literal; }
class GvRef        : PropertyValue { public string Name; }   // → ctx.Gv[name]
class FormulaValue : PropertyValue { public string Expr; }    // → ctx.Eval(Expr)
```

### 文本内联
```csharp
string ResolveText(string tmpl, EvalContext ctx) {
    // 匹配 $...$ 段 → 求值 → 替换；非公式段原样保留
    return Regex.Replace(tmpl, @"\$(.+?)\$", m => ctx.Eval(m.Groups[1].Value).ToString());
}
```

### 类型相容
- 期望 `NUMBER`：公式返回非数 → 回退 0 / 上值（见 P2-05 容错）。
- 期望 `COLOR`：公式返回字符串 `#RRGGBB` → 解析 `Brush`；失败回退默认色。

## 技术选型
C# 多态 + `System.Text.RegularExpressions`（`技术栈选型.md` §6 公式 / 原子属性行）。无外部依赖。

## FR 映射
功能需求 属性三元组（静态 / gv / 公式）；文本内联公式（KLWP 风格，见 `klwp/Kustom 原子模型.md`）。

## 验收
Text 设 Formula `$df(hh:mm:ss)$` → 实时；设 GvRef `gv:clock_fmt` → 随 gv 变；内联 `"电量 $bi(level)$%"` → 正确插值；类型不匹配不崩、回退默认。

## 依赖与顺序
- 依赖：`P2-03`(Eval)、`P2-04`(Gv)。
- 被依赖：P2-01 六类原子属性、P2-05 增量重算（脏标记基于三元组变更）、P2-06 持久化（三元组序列化）。

## 风险 / 开放
- 三元组循环引用（gv→公式→gv）需检测防死循环（P2-05 容错一并处理）。
- 内联 `$` 转义（字面量含 `$`）需约定转义规则（如 `\$`）。
