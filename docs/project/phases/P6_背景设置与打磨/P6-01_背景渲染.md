# P6-01 背景渲染

> Phase：P6 · 背景 + 设置面板 + 持久化闭环 + 打磨
> 上游：`v1开发计划.md` P6、`功能需求.md` §4 背景、FR-BG-01~05、`技术栈选型.md` §6（背景）
> 关联：`P1-01_多层模型.md`（WallpaperLayer）、`P3-03_页面.md`（每页独立背景）、`P6-02_设置面板.md`

## 目标
为（最底）壁纸层 / 每页提供背景渲染：图片（填充 / 适应 / 平铺 / 居中 / 拉伸）、纯色、线性 · 径向渐变，以及 `dwmapi` 亚克力毛玻璃；每页独立背景 + 切换过渡。

## 范围
**包含**
- 图片：`ImageBrush` + `Stretch` 五模式（Fill/Uniform/UniformToFill/Tile/Center，对应需求 填充/适应/平铺/居中/拉伸）。
- 纯色：`SolidColorBrush`。
- 渐变：`LinearGradientBrush` / `RadialGradientBrush`。
- 毛玻璃：`DwmEnableBlurBehindWindow`（P/Invoke `dwmapi`）实现亚克力模糊（v1.2 范围，v1.0 最小集仅图片/纯色/渐变，见 `版本排期.md` §2）。
- 每页独立背景：绑定 `Page.Background`（接 `P3-03`）。
- 切换过渡：页面 / 层切换时背景淡入淡出（复用 `P5-01` Storyboard）。

**不含**：动态壁纸 / 视频（FR-BG-06，v2）；网页背景（v2 WebView2）。

## 关键设计
### 渲染
```csharp
class BackgroundRenderer {
    UIElement Render(Background bg) => bg.Kind switch {
        Image  => new Rectangle { Fill = new ImageBrush(img) { Stretch = Map(bg.Mode) } },
        Solid  => new Rectangle { Fill = new SolidColorBrush(bg.Color) },
        Linear => new Rectangle { Fill = new LinearGradientBrush(...) },
        Radial => new Rectangle { Fill = new RadialGradientBrush(...) },
    };
}
```

### 毛玻璃（亚克力）
```csharp
// P/Invoke dwmapi.dll
[DllImport("dwmapi")] static extern int DwmEnableBlurBehindWindow(IntPtr hWnd, ref DWM_BLURBEHIND p);
// 设 p.fEnable=1, p.hRgnBlur=CreateRectRgn(0,0,width,height) → 窗口客户区模糊穿透
```
- 仅壁纸层（最底）启用毛玻璃时，其下原生桌面被模糊显示 → 亚克力效果。
- 需 `Window` 透明 + `AllowsTransparency`（P0 已设）。

## 技术选型
WPF `ImageBrush` / `GradientBrush` + `dwmapi` P/Invoke（`技术栈选型.md` §6 背景行）。无第三方。

## FR 映射
FR-BG-01（图片五模式）、FR-BG-02（纯色）、FR-BG-04（渐变）、FR-BG-03/05（毛玻璃 + 切换过渡，v1.2）。

## 验收
壁纸层显示图片（五模式正确）/ 纯色 / 渐变；v1.2 毛玻璃生效（原生桌面模糊）；切页背景淡变；每页背景独立。

## 依赖与顺序
- 依赖：`P1-01`(WallpaperLayer)、`P3-03`(页背景)、`P5-01`(过渡)。
- 被依赖：P6-02 设置面板配置背景、P6-03 持久化背景参数。

## 风险 / 开放
- 毛玻璃性能：全屏 `DwmEnableBlurBehindWindow` 在低端 GPU 有开销，需 `P2-05` 降频 / 静态时停更。
- 图片内存：大图需 `DecodePixelWidth` 限制解码尺寸守 NFR-01。
- 多屏（v2）每屏独立背景。
