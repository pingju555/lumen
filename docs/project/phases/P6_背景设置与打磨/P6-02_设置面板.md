# P6-02 设置面板

> Phase：P6 · 背景 + 设置面板 + 持久化闭环 + 打磨
> 上游：`v1开发计划.md` P6、`功能需求.md` §7 设置 / 控制层、FR-SET-01~07、`技术栈选型.md` §6（设置面板）
> 关联：`P0-02_守护进程.md`（自启开关）、`P3-01_网格.md`（间距）、`P6-01_背景渲染.md`、`P6-03_持久化闭环.md`（导入导出）

## 目标
提供 WPF 设置面板（托盘图标唤起），配置：显隐快捷键、网格间距四档、背景、自启、多屏（v2 占位）、配置导入导出。面板为**独立 `Topmost=True` 子窗口**（不污染覆盖层，且浮于其上，见 `技术栈选型.md` §6 设置面板行 / MEMORY 关键坑）。

## 范围
**包含**
- 托盘：`Hardcodet.NotifyIcon.Wpf` 托盘图标 → 左键 / 右键菜单唤设置、退出（走 `P0-02` clean-shutdown）。
- 设置项：
  - 显隐快捷键（注册 `RegisterHotKey`，接 P0 预留）。
  - 网格间距四档（20/40/60/120，写 `P3-01` 配置）。
  - 背景（图片 / 纯色 / 渐变，写 `P6-01`）。
  - 自启开关（`Autostart.SetEnabled(bool)`：读 `Autostart.Enabled` / 写注册表；**正常用户走此开关**。`--install`/`--uninstall` 命令行仅作调试 / 部署入口，不面向终端用户）。
  - 多屏（v2 占位 UI）。
- 导入 / 导出配置（JSON，接 `P6-03`）。

**不含**：编辑器内实时拖拽布局（v1 用画布直接编辑，设置仅参数）；账户云同步（v2）。

## 关键设计
### 托盘 + 窗口
```csharp
var icon = new TaskbarIcon { Icon = ..., ContextMenu = menu };
icon.TrayMouseDoubleClick += (_,_) => ShowSettings();   // 独立 Window Topmost=True
```
- 设置窗口：`Topmost=True` 独立实例（非覆盖层子元素），保证浮于覆盖层及所有普通窗口之上调试可见（MEMORY 关键坑：覆盖层主窗**不**置顶，设置窗才置顶）。

### 双向绑定
- 设置项 ←→ 内存 `Config` 模型（MVVM 轻量，手写 `INotifyPropertyChanged`）。
- 改网格间距 → 即时 `P3-01` 重排 + 持久化（`P6-03`）。

## 技术选型
`Hardcodet.NotifyIcon.Wpf`（NuGet，`技术栈选型.md` §5 白名单）、WPF `Window` + MVVM 手写（`技术栈选型.md` §6 设置面板行）。守 NFR-01 体积（轻量包）。

## FR 映射
FR-SET-01（快捷键）、FR-SET-02（网格间距）、FR-SET-03（背景）、FR-SET-04（自启）、FR-SET-05/06/07（多屏 / 导入导出 / 其它）。

## 验收
托盘唤出设置；改网格间距即时生效；改背景即时生效；自启开关同步注册表；导入导出配置无丢失。

## 依赖与顺序
- 依赖：`P0-02`(自启/退出)、`P3-01`(间距)、`P6-01`(背景)、`P6-03`(导入导出)。
- 被依赖：P6-03 持久化闭环（设置写盘）、P6-04 打磨（NFR 抽检）。

## 风险 / 开放
- `Hardcodet.NotifyIcon.Wpf` 在 .NET 8 兼容性需验证（社区维护版），若弃用可换 `NotifyIcon` 手写 P/Invoke（体积更优）。
- 设置窗 `Topmost=True` 与系统对话框 z 序：仅设置窗用，勿给覆盖层主窗（`技术栈选型.md` MEMORY 关键坑）。
