# P3-03 页面

> Phase：P3 · 网格 + 小组件 + 页面
> 上游：`v1开发计划.md` P3、`功能需求.md` §6（页面）、FR-PAGE-01~06、`技术栈选型.md` §6（页面）
> 关联：`P1-01_多层模型.md`、`P3-01_网格.md`、`P3-02_小组件.md`、`P2-06_持久化.md`

## 目标
支持多页面工作区：每页独立网格 + 背景 + 原子 / 小组件；页面间切换（指示器 / 滚轮 / 快捷键）+ 淡入淡出过渡；页面管理（增删改名排序），上限默认 9。

## 范围
**包含**
- `Page` 模型：`ObservableCollection<Layer>`（或独立 LayerStack）+ 网格参数 + 背景引用 + 名称。
- `ObservableCollection<Page>` 驱动 UI；切换即换当前页的 LayerStack 挂载。
- 切换方式：底部指示器点击 / 鼠标滚轮 / 全局快捷键（接入 `P0-02` 快捷键预留）。
- 过渡：`Storyboard` 淡入淡出（旧页淡出 + 新页淡入）。
- 管理：新增 / 删除 / 重命名 / 排序（拖拽列表）。
- 上限：默认 9 页（可配置）。

**不含**：页面间共享层（v1 每页独立）；页面同步（多设备，v2 外部）。

## 关键设计
### 模型
```csharp
class Page {
    public string Name;
    public LayerStack Layers = new();      // 该页独立层栈
    public double GridSize;                // 该页网格档位
    public BackgroundRef Background;        // P6-01
}
class PageManager {
    public ObservableCollection<Page> Pages = new();
    public int Current;
    public void SwitchTo(int i);           // 换挂载 + 过渡动画
}
```

### 切换 + 过渡

- 指示器：`ItemsControl` 绑定 `Pages`，当前页高亮。
- 滚轮 / 快捷键 → `SwitchTo(Current ± 1)`（循环或夹紧）。

## 技术选型
WPF `ObservableCollection` + `Storyboard`（`技术栈选型.md` §6 页面行）。无第三方。

## FR 映射
FR-PAGE-01~06（多页 / 独立网格背景 / 切换方式 / 过渡 / 管理 / 上限 9）。

## 验收
多页各自独立网格 + 背景；滚轮 / 点击指示器切换无闪烁（淡入淡出）；增删改名排序生效；超 9 页禁止新增。

## 依赖与顺序
- 依赖：`P1-01`(层)、`P3-01`(网格参数)、`P3-02`(页内 widget)、`P2-06`(page DTO)。

## 风险 / 开放

- 内存：9 页全量原子常驻可能涨内存，v1 页数少可接受，v2 可懒加载。
