# P3-01 网格

> Phase：P3 · 网格 + 小组件 + 页面
> 上游：`v1开发计划.md` P3、`功能需求.md` FR-GRID-01~12、`技术栈选型.md` §6（网格）
> 关联：`P1-02_坐标系.md`、`P1-04_Px画布与Text原子.md`、`P3-02_小组件.md`、`P2-06_持久化.md`

## 目标
在 `GridLayer` 上实现 gcd 间距网格：四档 {20,40,60,120}（1920×1080），cell↔px 换算，原子 / 小组件吸附到 cell，支持跨行列 resize 与冲突禁止。

## 范围
**包含**
- gcd 间距推导：基于工作区尺寸求最大公约数间距，提供四档可选。
- `GridLayer` 用 `UniformGrid`（或自定义 panel）按 gridSize 划分 cell。
- 吸附：拖拽落点 snap 到最近 cell 原点（`P1-04` 的 snapPx 升级为 cell）。
- 跨行列 resize：小组件可占多 cell，resize 时按 cell 对齐。
- 冲突禁止：同 cell 已被占 → 禁止放置（或堆叠需显式 Container）。

**不含**：网格视觉样式细节（线宽 / 颜色在设置，P6）；小组件定义（P3-02）。
