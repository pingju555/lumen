# P3-04 预设库

> Phase：P3 · 网格 + 小组件 + 页面
> 上游：`v1开发计划.md` P3、`功能需求.md` FR-LAYER-08~10、`技术栈选型.md` §6（预设）
> 关联：`P1-01_多层模型.md`（三类预设）、`P3-01_网格.md`、`P3-02_小组件.md`、`P3-03_页面.md`

## 目标
提供**预设库**：内置固定预设（三类层：壁纸层 / 网格化层 / 原子化 Px 画布层）+ 用户自定义预设（增删改导入导出）。预设即"层 / 网格 / 背景 / 小组件组合"的模板，一键套用复用。

## 范围
**包含**
- 内置预设：`WallpaperOnly` / `GridWorkspace` / `CanvasFree` 三类（对应 FR-LAYER-03 定义），含默认层栈。
- 用户预设：`Preset` = 命名 + 层栈模板 + 网格参数 + 关联小组件；增删改。
- 导入导出：预设序列化 JSON（复用 `P2-06` 机制），便于分享。
- 套用：选预设 → 重建当前页 / 项目的层栈。

**不含**：预设在线市场（v1 本地）；复杂条件预设（v1 静态模板）。

## 关键设计
### 预设模型
```csharp
class Preset {
    public string Name;
    public LayerKind[] Layers;             // 三类预设组合
    public double GridSize;                // 网格档位（GridWorkspace 用）
    public List<string> WidgetRefs;        // 关联小组件 id
    public BackgroundRef Background;       // 背景（P6-01）
}
static class PRESET_LIBRARY {
    Dictionary<string, Preset> _builtin;   // 三类固定
    Dictionary<string, Preset> _user;      // 用户增删
    public Preset Get(string name);
    public void Apply(string name, Project proj);  // 重建层栈
    public void Import(Stream json); public void Export(string name, Stream out);
}
```

### 三类内置映射
- `WallpaperOnly` → 仅 `WallpaperLayer`（最底背景）。
- `GridWorkspace` → `WallpaperLayer` + `GridLayer`（默认 40px 网格）。
- `CanvasFree` → `WallpaperLayer` + `CanvasLayer`（自由 px）。

## 技术选型
C# 集合 + `System.Text.Json` 导入导出（`技术栈选型.md` §6 预设行）。无第三方。

## FR 映射
FR-LAYER-08（预设类型）、FR-LAYER-09（用户自定义增删改）、FR-LAYER-10（导入导出）。

## 验收
一键套用三类内置预设 → 层栈正确重建；自建预设可增删改并导出 JSON、再导入复用。

## 依赖与顺序
- 依赖：`P1-01`(层类型)、`P3-01`(网格参数)、`P3-02`(widget 关联)、`P2-06`(序列化)。
- 被依赖：P6-02 设置面板套用预设、P3-03 页面组合（预设可含多页模板）。

- 预设版本：导入旧版预设需经 `P2-06` 的 `Migrate`。
