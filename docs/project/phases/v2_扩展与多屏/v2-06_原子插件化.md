# v2-06 原子插件化（动态加载）

> 版本：v2.0 · 扩展与多屏（deferred 分支）
> 上游：`版本排期.md` §5、`功能需求.md` FR-ATOM-11
> 关联：`P1-03_Atom抽象与注册.md`（编译期注册 → 插件式）、`P6-03_持久化闭环.md`（插件清单）

## 目标
将原子从"编译期注册"升级为"插件式动态加载"：第三方 / 用户可编写原子 dll，运行时经 `AssemblyLoadContext` 加载并注册到 `ATOM_REGISTRY`，无需重编译主程序。

## 范围
**包含**
- `IAtomPlugin` 接口：插件 dll 实现 → 暴露 `Atom` 子类 + 元数据（名称 / 版本 / 依赖）。
- 加载器：`AssemblyLoadContext`（隔离）加载插件 dll → 反射取 `IAtomPlugin` → `ATOM_REGISTRY.Register`。
- 插件清单：配置列出启用插件（存 `P6-03`）。
- 卸载（可选）：ALC 卸载需无引用残留，v1 插件可仅加载不卸载（重启生效）。

**不含**：插件市场 / 在线下载（v1 本地 dll）；插件沙箱深度隔离（v1 基础 ALC 隔离）。

## 关键设计
```csharp
interface IAtomPlugin {
    string Name { get; }
    IEnumerable<Type> AtomTypes { get; }     // 派生自 Atom
}
class PluginLoader {
    AssemblyLoadContext _ctx = new("atoms", isCollectible: true);
    void Load(string dll) {
        var asm = _ctx.LoadFromAssemblyPath(dll);
        foreach (var p in asm.GetTypes().Where(t => typeof(IAtomPlugin).IsAssignableFrom(t)))
            ATOM_REGISTRY.RegisterPlugin((IAtomPlugin)Activator.CreateInstance(p));
    }
}
```

## 技术选型
`System.Reflection` + `AssemblyLoadContext`（.NET 原生插件机制，`技术栈选型.md` 插件化行）。零 NuGet。

## FR 映射
FR-ATOM-11（原子插件化动态加载）。

## 验收
放置第三方原子 dll → 重启加载 → 新原子出现在注册表并可渲染；主程序无需重编译。

## 依赖与顺序
- 依赖：`P1-03`(注册表抽象)、`P6-03`(插件清单持久化)。
- 被依赖：v2 生态扩展（社区原子）。

## 风险 / 开放
- 安全：加载任意 dll 有执行任意代码风险，v1 仅加载用户明示可信路径。
- 版本兼容：插件引用主程序接口版本需契约稳定（公共接口独立 assembly 更佳）。
- ALC 卸载：`isCollectible` 卸载需确保无存活原子引用，否则内存泄漏；v1 可放弃卸载。
