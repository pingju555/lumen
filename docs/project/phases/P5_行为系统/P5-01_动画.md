# P5-01 动画

> Phase：P5 · 行为系统（动画 / 触发器 / 按钮）
> 上游：`v1开发计划.md` P5、`功能需求.md` §1.7 行为系统、FR-BEH-*`、`技术栈选型.md` §6（动画）
> 关联：`P5-02_触发器.md`、`P5-03_按钮行为.md`、`P2-05_增量重算与容错.md`

## 目标
为原子 / 容器提供动画能力：进入 / 退出 / 循环三类，含淡入 · 滑动 · 缩放 · 旋转，支持缓动（easing）与延迟（delay）。动画由触发器或按钮动作触发。

## 范围
**包含**
- `AnimationSpec`：类型（Fade/Slide/Scale/Rotate）、方向、时长、缓动函数、延迟、触发点（Enter/Exit/Loop）。
- 附着：绑定到原子 `UIElement`，由 `Storyboard` 驱动属性（Opacity / Translate / Scale / RotateTransform）。
- 循环：Loop 动画持续运行（如时钟刻度旋转），受 `P2-05` 降频约束（失焦暂停）。
- 进入 / 退出：显隐切换、页面切换（接 `P3-03`）、触发器激活时播放。

**不含**：复杂关键帧路径动画（v1 基础四型）；物理模拟（v1 不做）。

## 关键设计
### 规格
```csharp
enum AnimKind { Fade, Slide, Scale, Rotate }
class AnimationSpec {
    public AnimKind Kind;
    public AnimPhase Phase;        // Enter / Exit / Loop
    public TimeSpan Duration = TimeSpan.FromSeconds(0.3);
    public string Easing = "CubicOut";
    public TimeSpan Delay;
    public double From, To;        // 视 Kind：Opacity 0→1 / Translate / Scale / Angle
}
```

### 播放
```csharp
class AnimationPlayer {
    public Storyboard Build(UIElement target, AnimationSpec spec) {
        // 依 Kind 创建 DoubleAnimation on Opacity / RenderTransform 分量
        // 设 EasingFunction + BeginTime = Delay
    }
    public void Play(UIElement t, AnimationSpec s) => Build(t,s).Begin();
}
```
- Enter：元素加入可视树时 `Play`；Exit：移除前播放完再卸载（或淡出即隐藏）。
- Loop：完成后 `RepeatBehavior=Forever`；失焦经 `P2-05` 暂停 `CompositionTarget.Rendering`。

## 技术选型
WPF `Storyboard` + `DoubleAnimation` + `EasingFunction`（原生，`技术栈选型.md` §6 动画行）。无第三方。

## FR 映射
FR-BEH-（动画相关，§1.7 行为系统）。

## 验收
新原子淡入出现；按钮触发缩放动画；循环旋转在失焦后暂停、回焦恢复；缓动 / 延迟生效。

## 依赖与顺序
- 依赖：`P2-05`(降频暂停)、`P3-03`(页面切换过渡复用)。
- 被依赖：`P5-02`(触发器触发动画)、`P5-03`(按钮动作触发动画)。

## 风险 / 开放
- 大量 Loop 动画的 CPU：需 `P2-05` 统一停绘；v1 动画量可控。
- 退出动画与即时显隐切换冲突：需定义"切换是否等退出播完"（默认淡出即切，可选等完）。
