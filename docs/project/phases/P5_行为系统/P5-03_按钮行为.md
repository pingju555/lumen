# P5-03 按钮行为

> Phase：P5 · 行为系统（动画 / 触发器 / 按钮）
> 上游：`v1开发计划.md` P5、`功能需求.md` §1.7 行为系统、FR-BEH-01~10、`技术栈选型.md` §6（按钮行为）
> 关联：`P5-01_动画.md`、`P5-02_触发器.md`、`P4-03_启动坞ai_an.md`（Launch）、`P3-03_页面.md`（切页）

## 目标
为按钮（可绑定到任意原子的点击，或 `Container`/`Icon` 充当按钮）定义行为动作：切层显隐 · 切页 · 运行程序 · 发快捷键 · 触发动画 · 开 URL · 执行脚本。动作可由 `P5-02` 触发器或用户点击驱动。

## 范围
**包含**
- `Action` 类型：
  - `ToggleLayer` / `ShowLayer` / `HideLayer`（层显隐）。
  - `SwitchPage`（接 `P3-03`）。
  - `RunApp`（桌面 exe，`Process.Start`；UWP 经 `P4-03.Launch`）。
  - `SendKeys`（快捷键，`System.Windows.Forms.SendKeys` 或 Win32）。
  - `PlayAnimation`（接 `P5-01`）。
  - `OpenUrl`（`Process.Start(url)` 默认浏览器）。
  - `RunScript`（执行脚本文件 / 命令，v1 限本地脚本，安全提示）。
- 按钮绑定：原子 `IsHitTestVisible=True` 接收点击 → 执行绑定 `Action` 列表。

**不含**：远程命令 / 网络控制（v1 本地）；脚本沙箱（v1 直接执行，需用户确认来源）。

## 关键设计
### 动作
```csharp
enum ActionKind { ToggleLayer, ShowLayer, HideLayer, SwitchPage,
                  RunApp, SendKeys, PlayAnim, OpenUrl, RunScript }
class Action {
    public ActionKind Kind;
    public Dictionary<string,object> Args;   // layerId / pageIndex / path / keys / url / script
}
class ActionExecutor {
    public void Execute(Action a) {
        switch (a.Kind) {
            case ActionKind.RunApp: Process.Start((string)a.Args["path"]);
            case ActionKind.OpenUrl: Process.Start((string)a.Args["url"]);
            case ActionKind.SendKeys: SendKeys.Send((string)a.Args["keys"]);
            case ActionKind.SwitchPage: _pages.SwitchTo((int)a.Args["idx"]);
            case ActionKind.PlayAnim: _anim.Play(target, spec);
            // ToggleLayer/ShowLayer 经 LayerStack
        }
    }
}
```

### 点击绑定
```csharp
// 原子可带 OnClick: List<Action>
ui.MouseDown += (_,_) => { foreach (var act in atom.OnClick) _exec.Execute(act); };
```

## 技术选型
`System.Diagnostics.Process`、`System.Windows.Forms.SendKeys`（或 Win32 `keybd_event`）、WPF 事件（`技术栈选型.md` §6 按钮行为行）。`SendKeys` 需引用 `System.Windows.Forms`（轻量）。

## FR 映射
FR-BEH-01~10（按钮行为全）。

## 验收
点击启动坞图标启动程序（`RunApp`/`P4-03.Launch`）；点按钮切页（`SwitchPage`）；空闲触发器（P5-02）驱动 `ToggleLayer` 显隐；点按钮发快捷键（`SendKeys`）控制其他程序。

## 依赖与顺序
- 依赖：`P3-03`(切页)、`P4-03`(Launch)、`P5-01`(动画)、`P5-02`(触发器驱动)。
- 被依赖：P6-02 设置面板配置按钮、便签/文件速览交互（v1.2）。

## 风险 / 开放
- `SendKeys` 焦点问题：需目标窗口聚焦，可能需 `SetForegroundWindow` 前置；或改用 Win32 `PostMessage`。
- 脚本执行安全：v1 仅执行用户显式配置的本地脚本，避免任意远程脚本。
- 多动作顺序：默认顺序执行，需要延迟/条件时接 P6 工作流（FR-WF）。
